Whenever we want to write a program, we should first think about writing comments.


What are comments?

Comments are description about the features of a program.
This means whatever we write in a program should be described using comments.

How many types of comments are available in java?

There are three types of comments available in Java:

  1. Single line commnets:

    These comments starts with "//" (double forward slashes). These comments should be written in single line

    Eg: //comments goes here

  2. Multi line commnets:

    These comments starts with "/*" (one forward slash followed by an one astrik) and ends with "*/" (one astrik followed by an forward slash). These comments are used to write in multiple lines

    Eg:
    /* This is a multi line comment. This is line one
    This is line two of the comment.
    This is line three of the comment.
    */

  3. Java documentation commnets:

    These comments starts with "/**" and ends with "*/". These comments are used to provide description for every feature in a Java Program

    Eg:

    /** description about a class */
    class DemoProgram{


    /** description about a method*/
    public static void main(String args[]){ }

    }
